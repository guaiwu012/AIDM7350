# 使用说明

这个版本把原来的单页网站拆成了：
- `index.html`：主页面，Section 5 只保留概括内容
- `response.html`：单独的 Response 详细页面
- `styles.css`：统一样式
- `script.js`：进度条、滚动显现、导航高亮、移动端菜单

## 你现在可以怎么用
1. 直接双击打开 `index.html`
2. 在 Section 5 点击按钮跳到 `response.html`
3. 若要继续精简，只需要改这两个文件：
   - 首页概括：`index.html`
   - 详细 response：`response.html`

## 建议你后面自己替换的内容
- 首页和 response 页的配图
- Response 页 references 区的最终引用格式
- 少量用词，让全站语气更统一
